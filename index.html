About

A monthly program for California-based media artists — working with light, code, screens, data, sound — to show work that isn't finished yet. Prototypes, experiments, tests. The kind of work that usually stays in the studio.

The work doesn't have to be resolved. It has to be alive.

Principles
  • Process over polish. The work doesn't need to be done. It needs to be honest.
  • Artist-first. The programming serves the artists, not the market.
  • Local roots. California-based. Building from here.
  • Intimate scale. Small audiences, real conversation.
